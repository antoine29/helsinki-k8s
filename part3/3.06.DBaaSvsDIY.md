| | DB as service   | Do it yourself   |
|-------------- | -------------- | -------------- |
| Setup   | The easier option. All setup and configuration relies on the provider     | Difficulty it's up to you. Depending on the solution you'll pick, the difficulty will vary.    |
| Scaling    | Scaling process is managed by the provider     | You're in charge of scaling any resource needed.     |
| Billing    | Expensiver. Managed dbs (as any SaaS) could easily become greedy on money terms | Usually cheaper. You only pay for what you set up |
| Vendor locking    | Depending on the provided solution. | Pretty unlikely. As long as you're in charge, you'll be able to pivot to other solution |
